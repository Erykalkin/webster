# Project-Scoped Rules

## Interactive Verification and Consent Rule
- Для всех последующих редактирований (если не запрошено обратное), на каждом шагу, где необходимо использовать какие-либо данные, делать новые утверждения, выводы или вводить новые физические/математические параметры, агент обязан остановить работу, запросить подтверждение/уточнение у пользователя и продолжить только после получения ответа.
