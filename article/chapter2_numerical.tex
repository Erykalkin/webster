\chapter{Численные методы моделирования акустических систем}
\label{ch:numerical}

В данной главе описываются численные методы решения уравнений одномерной акустики, реализованные в рамках разработанного программного комплекса: метод конечных разностей во временной области (FDTD) для уравнения Вебстера и метод матриц длинных линий (TLM) в цилиндрической формулировке.

\section{Конечно-разностный метод (FDTD)}
\label{sec:fdtd}

Для решения нестационарного уравнения Вебстера~\eqref{eq:webster_time} применяется конечно-разностная схема второго порядка точности по времени и по пространству (схема типа «чехарда» или leapfrog).

Дискретизация расчетной области по пространству проводится с шагом $\Delta x$, по времени --- с шагом $\Delta t$. Обозначим значение давления в узле сетки $x_i = i\Delta x$ в момент времени $t^n = n\Delta t$ как $p_i^n$. Площадь сечения в узлах обозначается как $A_i$.

Аппроксимация пространственной производной в уравнении~\eqref{eq:webster_time} с учетом переменного сечения $A(x)$ записывается в виде:
\begin{equation}
    \left[ \frac{1}{A} \frac{\partial}{\partial x} \left( A \frac{\partial p}{\partial x} \right) \right]_i^n \approx \frac{1}{A_i} \frac{1}{\Delta x^2} \left[ A_{i+1/2} (p_{i+1}^n - p_i^n) - A_{i-1/2} (p_i^n - p_{i-1}^n) \right],
\end{equation}
где площади на полуцелых шагах аппроксимируются как среднее арифметическое: $A_{i \pm 1/2} = (A_i + A_{i \pm 1}) / 2$.

Разностная аппроксимация второй производной по времени имеет вид:
\begin{equation}
    \frac{\partial^2 p}{\partial t^2} \approx \frac{p_i^{n+1} - 2p_i^n + p_i^{n-1}}{\Delta t^2}.
\end{equation}

Подставляя данные аппроксимации в исходное уравнение, получаем явную схему для вычисления давления на новом временном слое $n+1$:
\begin{equation}
    p_i^{n+1} = 2p_i^n - p_i^{n-1} + \left(\frac{c \Delta t}{\Delta x}\right)^2 \frac{1}{A_i} \left[ A_{i+1/2} (p_{i+1}^n - p_i^n) - A_{i-1/2} (p_i^n - p_{i-1}^n) \right].
\end{equation}

Для устойчивости данной схемы необходимо выполнение условия Куранта--Фридрихса--Леви (CFL):
\begin{equation}
    CFL = \frac{c \Delta t}{\Delta x} \le 1.
\end{equation}

\section{Метод матриц длинных линий (TLM)}
\label{sec:tlm}

Метод матриц длинных линий (Transmission Line Matrix, TLM) во временной области основан на физической аналогии между распространением акустических волн и электрических импульсов в длинных линиях связи~\cite{johns1971}. Волновод представляется в виде цепочки элементарных линий задержки (линков), соединенных в узлах рассеяния (junctions).

В одномерном акустическом TLM-решателе для цилиндрических сегментов каждый линк характеризуется волновым сопротивлением $Z_i = \rho_0 c / A_i$. В каждом узле происходит рассеяние падающих импульсов давления $V^i$ (инцидентных волн) и формирование отраженных импульсов $V^r$.

Связь между падающими и отраженными импульсами на границе двух сегментов с импедансами $Z_1$ и $Z_2$ определяется коэффициентами отражения $R$ и прохождения $T$:
\begin{equation}
    R = \frac{Z_2 - Z_1}{Z_2 + Z_1}, \quad T = 1 + R = \frac{2 Z_2}{Z_2 + Z_1}.
\end{equation}

Процесс вычислений на каждом шаге по времени разделяется на две стадии:
\begin{enumerate}
    \item \textbf{Рассеяние (Scattering):} В каждом узле вычисляются отраженные импульсы на основе падающих.
    \item \textbf{Перенос (Advection / Propagation):} Отраженные импульсы распространяются в соседние узлы, становясь падающими на следующем временном шаге.
\end{enumerate}

\section{Архитектура программного комплекса}
\label{sec:architecture}

Разработанное программное обеспечение имеет гибридную архитектуру:
\begin{itemize}
    \item \textbf{Вычислительное ядро (C++):} Высокопроизводительные решатели TLM и FDTD реализованы в виде C++ шаблонов (заголовочные файлы \texttt{vt\_cylinder\_tlm\_solver.hpp} и \texttt{vt\_geometry\_simple.hpp}), что обеспечивает максимальную скорость расчетов во временной области.
    \item \textbf{Интерфейсный слой (Python):} Скрипты обертки (\texttt{vt\_all\_solvers\_wrapper.py}) осуществляют компиляцию C++ кода «на лету» под целевую платформу (Windows/Linux/macOS), генерируют файлы геометрии, запускают расчеты и импортируют результаты в формате CSV.
    \item \textbf{Анализ и визуализация (Jupyter/Python):} Модули визуализации используют библиотеку \texttt{matplotlib} с разработанными стилями оформления для построения графиков и сравнения спектров.
\end{itemize}
